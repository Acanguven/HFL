Tutorial

  1. First you need to register to the system using remote client. You can press the login button at top right

  2. You will be able to use your trial period only in one computer so take care!

  3. You just need to click download button at remote client and start the exe.

  4. Click smurf settings and add your smurfs.

  5. Click chat settings and set your chat messages, this is important. Without these messages you can be called botter.

  6. Press start button and have fun watching it.
